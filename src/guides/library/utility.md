# 工具类

## 配置相关

getConfig
获取插件配置
- getConfigValue 获取目标配置项的值
- getFullConfig 获取所有配置项的值

## 数据表相关

- getTables 获取数据表列表
- getFieldList 获取目标数据表字段列表
- getTablePrimaryKey 获取数据表主键字段名
- getFilterFields 获取需要过滤的字段列表

## 文件相关

- getFileInfo 获取文件信息
- getFileHeader 获取文件头信息
- getFileData 获取文件数据
- resolveFilePath 解析文件路径
- formatFileSize 格式化文件大小

## 其他

- generateFingerprint 生成数据指纹
- getConfiguration 获取导入预设配置详情
