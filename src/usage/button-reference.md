# 导入按钮引用

## 介绍

> 为了方便用户在不同控制器中使用导入功能，DataTrans 插件提供了一个可复用的导入按钮组件。通过引用该组件，可以快速在任何控制器的视图中添加导入按钮，并自动关联到预设配置和导入任务。

## 主要功能实现

> 导入按钮组件支持两种引用方式：仅前端引用和全栈引用。用户可以根据实际需求选择适合的方式来集成导入功能。

### 仅前端引用方式

> 通过在视图文件中直接引用导入按钮组件，可以快速添加导入功能，无需编写额外的后端代码。只需确保预设配置已正确设置，导入按钮即可正常工作。



### 全栈引用方式

> 通过在控制器中使用预配置调用方式，可以实现更灵活的导入功能集成。只需在控制器中调用预设配置，即可自动生成导入按钮并关联到相应的导入任务，适用于需要更复杂逻辑处理的场景。
