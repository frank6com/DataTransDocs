# 简介

> DataTrans 是基于 FastAdmin 开发的数据批量处理导入的插件扩展，支持多种数据格式的导入，提供强大的数据处理功能，异步导入过程理论上不受数据量限制，允许对数据导入进行预设配置并调用导入按钮快速增强其他模块的数据导入功能。

## 主要功能实现

### 1. **基于异步处理实现**：

通过消息队列分批异步处理数据，提升导入数据量及处理效率，减少服务器压力

### 2. **预设导入配置**：

支持对特定数据表设定预设导入规则，并提供灵活的配置选项，通过复用预设导入规则简化重复的导入设定操作

### 3. **任务跟踪**：
提供完整的导入任务跟踪功能，实时监控导入进度和结果，确保数据导入的可靠性

### 4. **支持环境检查**：

提供环境检查功能，确保服务器环境满足导入需求，避免导入失败

### 5. **快速集成** <Badge type="warning" text="高级功能" />：

提供灵活的预配置调用，可快速将导入功能集成到其他功能模块中，增强其他模块的数据导入能力

### 6. **数据批量预处理** <Badge type="warning" text="高级功能" />：

支持分批数据处理时进行数据的预处理转换，支持数据列的转换表达式，满足复杂的数据处理需求

### 7. **导入预览** <Badge type="warning" text="高级功能" />：

支持在正式导入前预览数据处理结果，帮助用户验证数据处理规则的正确性，避免导入错误数据
